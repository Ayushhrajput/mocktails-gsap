## üöÄ Getting Started

### Prerequisites
- Node.js (v16 or above recommended)
- npm or yarn package manager

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Ayushhrajput/mocktails-gsap.git
   cd mocktails-gsap
   ``` 

## Install dependencies

```bash
npm install
# or
yarn install
```
### Usage
```bash 
npm run dev
# or
yarn dev
```
### Deployment

```bash
npm run build
# or
yarn build
```
